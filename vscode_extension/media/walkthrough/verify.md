# 4. Verify the connection

You're done! Click the button below to:

- open the **Blender MCP** view in the activity bar
- run a `ping` against the add-on
- show the status bar item

If everything's wired up, the status bar shows **● Blender MCP** with a
millisecond ping in the tooltip. If not, the tooltip tells you exactly what's
wrong — token missing, Blender not running, or connection refused.

## What to try next

- Open Copilot Chat and ask: *"Create a red cube above a blue plane."*
- Open the **Take Viewport Screenshot** command for a live render preview.
- Browse **Recent Activity** in the Blender MCP sidebar — every command run
  is audit-logged with timing and arguments.

## Got stuck?

Run **Blender MCP: Show Output Channel** for diagnostic messages.
