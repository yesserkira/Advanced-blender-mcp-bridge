# Build a Red Chair

A sample prompt to test the Blender MCP Bridge end-to-end.
Paste this into your AI chat (Copilot, Claude, Cursor, Cline) and verify each step completes.

---

Build a simple red chair:

1. Create a cube for the seat, scale it to [1.5, 1.5, 0.1], position at [0, 0, 0.8]
2. Create 4 cylinder legs, each scaled to [0.1, 0.1, 0.8], positioned at the four corners
3. Create a cube for the backrest, scale to [1.5, 0.1, 0.8], position at [0, -0.7, 1.6]
4. Create a red material with metallic=0.1, roughness=0.6
5. Assign the material to all parts
6. Take a viewport screenshot to verify
